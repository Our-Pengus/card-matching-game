# 효과음 폴더

## 담당: 손아영

이 폴더에 게임 효과음 파일을 넣어주세요.

## 필요한 효과음

### 필수 효과음
- `click.mp3` - 카드 클릭 소리
- `match.mp3` - 짝 맞추기 성공 소리
- `mismatch.mp3` - 짝 맞추기 실패 소리
- `complete.mp3` - 게임 클리어 소리

### 선택 효과음
- `hover.mp3` - 카드 호버 소리
- `flip.mp3` - 카드 뒤집기 소리
- `combo.mp3` - 콤보 달성 소리
- `timesup.mp3` - 시간 초과 소리
- `bgm.mp3` - 배경 음악 (선택)

## 파일 요구사항

- **형식**: MP3 또는 WAV
- **길이**: 0.5~2초 (짧고 명확하게)
- **음량**: 적당히 조절 (너무 크지 않게)
- **용량**: 각 파일 500KB 이하 권장
