# 카드 이미지 폴더

## 담당: 손아영

이 폴더에 카드 앞면 이미지를 넣어주세요.

## 이미지 요구사항

- **형식**: PNG 또는 JPG
- **크기**: 200x280px (권장)
- **배경**: 투명 (PNG) 또는 흰색
- **개수**: 최소 15개 (지옥 난이도용)

## 임시 테스트 방법

이미지가 없어도 게임은 실행됩니다.
- 카드에 숫자(ID)가 표시됩니다
- 나중에 이미지를 추가하면 자동으로 적용됩니다
