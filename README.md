# 카드 매칭 게임

## 📋 프로젝트 개요

카드 매칭 게임은 짝을 맞추는 메모리 게임입니다. 다양한 난이도로 플레이할 수 있으며, 점수와 시간을 기록하여 실력을 향상시킬 수 있습니다.

## ✨ 주요 특징

- **5가지 난이도**: 하, 중, 상, 재앙, 지옥 - 각 난이도별 고유한 규칙과 컨셉
- **다양한 게임 요소**: 정답 짝 카드, 폭탄 카드, 시간 제한 등
- **점수 시스템**: 시도 횟수, 시간, 난이도에 따른 종합 점수 계산
- **풍부한 연출**: 호버 효과, 클릭 애니메이션, 성공/실패 피드백
- **난이도별 컨셉**: 각 난이도에 맞는 색감과 시각적 스타일

## 🎮 게임 기획

### 게임 목표
- 플레이어가 카드의 위치를 기억하고 짝을 맞추는 메모리 게임
- 난이도에 따라 다른 규칙과 도전 요소 제공
- 시간과 시도 횟수를 최소화하여 높은 점수 달성

### 핵심 메커니즘
- **카드 매칭**: 같은 카드 2장(또는 3장)을 찾아 짝 맞추기
- **정답 짝 카드**: 특정 조합을 찾으면 추가 점수 또는 보너스
- **폭탄 카드**: 클릭 시 페널티 (시간 감소, 카드 섞임 등)
- **즉사, 카드 섞임**: 특정 확률로 강한 패널티 적용
- **시간 제한**: 난이도에 따라 제한 시간 설정
- **점수 계산**: 시도 횟수, 시간, 난이도 가중치를 종합한 점수

### 전체 UX 플로우

#### 시작 화면
- 게임 제목 및 설명
- 난이도 선택 UI (지옥/상/중/하)
- 시작 버튼

#### 게임 화면
- 카드 그리드 배치
- 클릭 인터랙션 및 애니메이션
- 진행 상황 표시

#### 결과 화면
- 점수 표시
- 플레이 시간 표시
- 시도 횟수 표시
- 다시하기 버튼

### 게임 규칙

#### 난이도 옵션
- **지옥**: 최고 난이도 (카드 수 최대, 시간 제한)
- **상**: 높은 난이도
- **중**: 중간 난이도
- **하**: 쉬운 난이도 (카드 수 최소)

#### 점수 시스템
- 시도 횟수에 따른 점수 계산
- 시간에 따른 보너스/감점
- 난이도별 가중치 적용

### 연출 및 인터랙션

#### 시각 효과
- 카드 호버 효과
- 클릭 시 애니메이션
- 짝 맞춤 성공/실패 연출
- 난이도 선택 시 미리보기

#### 난이도별 컨셉
- **하**: 부드러운 색감, 친근한 느낌 (정답 짝 카드 2장, 카드 총 개수 8장, 4짝의 카드 제공, 제한 시간 ??분 or 기록 갱신)
- **중**: 밝은 색감, 균형잡힌 디자인 (정답 짝 카드 2장, 카드 총 개수 16장, 8짝의 카드 제공, 제한 시간 ??분 or 기록 갱신)
- **상**: 진한 색감, 도전적인 느낌 (정답 짝 카드 2장, 카드 총 개수 32장, 15짝의 카드 제공, 폭탄 카드 2장, 제한 시간?? 분 or 기록 갱신)
- **재앙**: 어두운 색감, 강렬한 연출 (정답 짝 카드 3장, 카드 총 개수 36장, 12짝의 카드 제공, 폭탄 카드 2장, 잘못 클릭 시 시간 추가 or 감소)
- **지옥**: 지옥 배경, 도전자를 깔보는 느낌의 대사 (정답 짝 카드 3장, 카드 총 개수 48장, 22짝의 카드 제공, 폭탄 카드 4장, 엄격한 제한 시간, 폭탄 카드 클릭 시 시간 대폭 감소 or 3%의 확률로 남은 카드 섞임, 1%의 확률로 즉사)
