# 카드 매칭 게임

## 📋 프로젝트 개요

카드 매칭 게임은 짝을 맞추는 메모리 게임입니다. 다양한 난이도로 플레이할 수 있으며, 점수와 시간을 기록하여 실력을 향상시킬 수 있습니다.

## ✨ 주요 특징

- **4가지 난이도**: 쉬움, 보통, 어려움, 지옥 - 각 난이도별 고유한 규칙과 컨셉
- **다양한 게임 요소**: 카드 매칭, 폭탄 카드, 시간 제한, 하트 시스템 등
- **점수 시스템**: 시도 횟수, 시간, 난이도에 따른 종합 점수 계산
- **풍부한 연출**: 호버 효과, 클릭 애니메이션, 성공/실패 피드백
- **난이도별 컨셉**: 각 난이도에 맞는 색감과 시각적 스타일
- **귀여운 UI/UX**: 파스텔톤 컬러, 둥근 모서리, 캐릭터 마스코트

## 🎮 게임 기획

### 게임 목표
- 플레이어가 카드의 위치를 기억하고 짝을 맞추는 메모리 게임
- 난이도에 따라 다른 규칙과 도전 요소 제공
- 시간과 시도 횟수를 최소화하여 높은 점수 달성

### 핵심 메커니즘
- **카드 매칭**: 같은 카드 2장(또는 3장)을 찾아 짝 맞추기
- **카드 미리보기**: 쉬움/보통/지옥 난이도에서 게임 시작 전 카드 공개
- **하트 시스템**: 실패 시 하트 감소, 하트가 0이 되면 게임 오버
- **히든 카드**: 보너스 카드 (매칭 성공 시 모든 카드 1초간 공개)
- **폭탄 카드**: 클릭 시 페널티 (시간 감소, 카드 섞임 등)
- **즉사, 카드 섞임**: 특정 확률로 강한 패널티 적용
- **시간 제한**: 난이도에 따라 제한 시간 설정
- **점수 계산**: 시도 횟수, 시간, 난이도 가중치를 종합한 점수

### 전체 UX 플로우

#### 시작 화면
- 게임 제목 및 설명
- 난이도 선택 UI (쉬움/보통/어려움/지옥)
- 각 난이도별 설명 및 미리보기
- 시작 버튼 (큰 둥근 버튼)
- 캐릭터 마스코트 배치

#### 게임 화면
- **상단 UI**
    - 좌측: 하트 카운터 (♥×n 형식)
    - 우측: 타이머 (시간 제한이 있는 난이도)
- **중앙**: 카드 그리드 배치
    - 둥근 모서리 카드 디자인
    - 파스텔톤 배경색
    - 클릭 인터랙션 및 애니메이션
- **하단**: 캐릭터 마스코트 고정 배치
- 진행 상황 표시

#### 게임 오버 화면 (하트 소진 시)
- **중앙 팝업 모달**
    - 부서진 하트 아이콘
    - "실패" 또는 "아쉬워요!" 메시지
    - 달성한 점수 표시
    - 맞춘 카드 짝 수 표시
    - 플레이 시간 표시
- **버튼 영역**
    - 재시도 버튼 (재생 아이콘)
    - 난이도 선택 버튼 (홈 아이콘)
- **배경**: 반투명 오버레이
- **하단**: 캐릭터 마스코트 유지

#### 게임 완료 화면 (모든 카드 매칭 성공 시)
- **중앙 팝업 모달**
    - 반짝이는 하트 아이콘 또는 별 아이콘
    - "성공!" 또는 "완벽해요!" 메시지
    - 최종 점수 표시
    - 플레이 시간 표시
    - 총 시도 횟수 표시
    - 남은 하트 표시
- **버튼 영역**
    - 다시하기 버튼
    - 난이도 선택 버튼
- **배경**: 반투명 오버레이
- **하단**: 캐릭터 마스코트 (기뻐하는 표정)

### 게임 규칙

#### 난이도 옵션
- **지옥**: 최고 난이도 (카드 수 최대, 시간 제한)
- **어려움**: 높은 난이도
- **보통**: 중간 난이도
- **쉬움**: 쉬운 난이도 (카드 수 최소)

#### 하트 시스템
- **쉬움**: 하트 5개
- **보통**: 하트 10개
- **어려움**: 하트 20개
- **지옥**: 하트 25개

#### 하트 감소 규칙
- 카드 매칭 실패 시: 하트 1개 감소
- 하트가 0이 되면: 게임 오버 화면으로 전환

#### 게임 종료 조건
- **게임 완료**: 모든 카드 짝을 맞추면 게임 완료 화면으로 전환
- **게임 오버**: 하트를 모두 소진하면 게임 오버 화면으로 전환
- **시간 초과** (어려움/지옥): 제한 시간 종료 시 게임 오버 화면으로 전환

#### 점수 시스템
- 시도 횟수에 따른 점수 계산
- 시간에 따른 보너스/감점
- 남은 하트에 따른 보너스
- 난이도별 가중치 적용

### 연출 및 인터랙션

#### 시각 효과
- 카드 호버 효과 (약간 떠오르는 효과)
- 클릭 시 뒤집기 애니메이션
- 짝 맞춤 성공 시: 카드 반짝임 + 소리 효과
- 짝 맞춤 실패 시: 카드 흔들림 + 하트 감소 애니메이션
- 난이도 선택 시 미리보기
- 하트 감소 애니메이션 (하트가 깨지는 효과)
- 게임 오버/완료 시 팝업 모달 슬라이드 인 애니메이션
- 캐릭터 마스코트 표정 변화 (성공/실패/대기)

#### UI/UX 디자인 가이드
- **컬러 팔레트**: 파스텔톤 (연두, 핑크, 라벤더, 피치, 민트 등)
- **타이포그래피**: 둥글고 귀여운 폰트
- **버튼**: 큰 둥근 버튼, 아이콘 포함
- **카드**:
    - 둥근 모서리 (border-radius: 16px~24px)
    - 흰색 테두리
    - 그림자 효과
    - 뒷면: 파스텔 단색
    - 앞면: 파스텔 배경 + 중앙 아이콘
- **모달**:
    - 둥근 모서리
    - 흰색 배경
    - 부드러운 그림자
    - 중앙 정렬
- **캐릭터 마스코트**:
    - 곰, 토끼, 고양이 등 귀여운 동물
    - 하단 우측 또는 중앙 고정
    - 상황별 표정 변화

#### 난이도별 컨셉
- **쉬움**: 부드러운 색감, 친근한 느낌 (연두색 계열)
    - 매칭 방식: 같은 카드 2장 찾기
    - 카드 구성: 총 10장 (4쌍 + 히든 2장)
    - 그리드: 4x2
    - 하트: 5개
    - 게임 시작: 2.5초간 모든 카드 공개 후 시작
    - 제한 시간: 60초
    - 배경색: 연한 그린 (#E8F5E9)

- **보통**: 밝은 색감, 균형잡힌 디자인 (옐로우 계열)
    - 매칭 방식: 같은 카드 2장 찾기
    - 카드 구성: 총 18장 (8쌍 + 히든 2장)
    - 그리드: 4x4
    - 하트: 10개
    - 게임 시작: 4초간 모든 카드 공개 후 시작
    - 제한 시간: 90초
    - 배경색: 연한 옐로우 (#FFF3E0)

- **어려움**: 진한 색감, 도전적인 느낌 (오렌지 계열)
    - 매칭 방식: 같은 카드 2장 찾기
    - 카드 구성: 총 34장 (15쌍 + 히든 2장 + 폭탄 2장)
    - 그리드: 8x4
    - 하트: 20개
    - 게임 시작: 미리보기 없음
    - 제한 시간: 90초
    - 특수 요소: 폭탄 카드 2장 (클릭 시 시간 감소)
    - 배경색: 연한 오렌지 (#FFEBEE)

- **지옥**: 지옥 배경, 도전적인 느낌 (레드/다크 계열)
    - 매칭 방식: 같은 카드 3장 찾기
    - 카드 구성: 총 66장 (19세트 × 3 = 57장 + 히든 3장 + 폭탄 6장)
    - 그리드: 11x6
    - 하트: 25개
    - 게임 시작: 5초간 모든 카드 공개 후 시작
    - 제한 시간: 60초
    - 폭탄 카드 효과:
        - 기본: 시간 대폭 감소
        - 3% 확률: 남은 카드 섞임
        - 1% 확률: 즉사 (게임 오버)
    - 배경색: 어두운 레드 (#FFCDD2)
    - 특수 연출: 불꽃 이펙트, 경고 메시지

### 카드 디자인

#### 테마
- **귀여운 아이콘 세트**
    - 동물: 고양이, 강아지, 토끼, 곰, 여우
    - 음식: 케이크, 아이스크림, 도넛, 컵케이크
    - 자연: 꽃, 나무, 별, 달, 구름
    - 사물: 로켓, 다이아몬드, 왕관, 하트, 별
    - 판타지: 유니콘, 무지개, 마법 지팡이

#### 난이도별 디자인 컨셉
- 각 난이도에 맞는 시각적 스타일 적용
- 카드 배경색 복잡도 조절
- 아이콘 복잡도 조절 (쉬움: 단순 / 지옥: 복잡)

### 스토리보드

#### 메시지 문구

**게임 진행 중**
- "카드를 클릭해보세요!"
- "좋아요!", "잘했어요!", "완벽해요!"
- "아쉬워요", "다시 한번!", "힘내세요!"

**게임 오버 화면**
- "실패" (큰 글씨)
- "아쉬워요! 다시 도전해보세요!"
- "하트를 모두 사용했어요"

**게임 완료 화면**
- "성공!" (큰 글씨)
- "완벽해요!"
- "축하합니다!"

**지옥 난이도 전용 (도발적 메시지)**
- "이 정도는 할 수 있겠지?"
- "벌써 포기?"
- "아직 멀었어요"
- "다음엔 잘할 수 있을 거예요... 아마도?"

#### 메시지 노출 타이밍
- 게임 시작 시 안내 메시지 (말풍선)
- 카드 미리보기 중 카운트다운 표시 (쉬움/보통 난이도)
- 카드 매칭 성공/실패 시 즉시 피드백 (작은 팝업)
- 하트 감소 시 시각적 피드백 (하트 깨지는 애니메이션)
- 하트 소진 시 게임 오버 모달 팝업
- 모든 카드 매칭 완료 시 게임 완료 모달 팝업
- 제한 시간 종료 시 게임 오버 모달 팝업

#### 캐릭터 마스코트 반응
- **대기 상태**: 편안한 표정, 가끔 깜빡임
- **성공 시**: 기쁜 표정, 손 흔들기
- **실패 시**: 아쉬운 표정, 위로하는 제스처
- **게임 오버**: 슬픈 표정, 응원하는 제스처
- **게임 완료**: 매우 기쁜 표정, 축하하는 제스처
