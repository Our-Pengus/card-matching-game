# 카드 매칭 게임

## 📋 프로젝트 개요

카드 매칭 게임은 짝을 맞추는 메모리 게임입니다. 다양한 난이도로 플레이할 수 있으며, 점수와 시간을 기록하여 실력을 향상시킬 수 있습니다.

## ✨ 주요 특징

- **4가지 난이도**: 쉬움, 보통, 어려움, 지옥 - 각 난이도별 고유한 규칙과 컨셉
- **다양한 게임 요소**: 카드 매칭, 폭탄 카드, 시간 제한 등
- **점수 시스템**: 시도 횟수, 시간, 난이도에 따른 종합 점수 계산
- **풍부한 연출**: 호버 효과, 클릭 애니메이션, 성공/실패 피드백
- **난이도별 컨셉**: 각 난이도에 맞는 색감과 시각적 스타일

## 🎮 게임 기획

### 게임 목표
- 플레이어가 카드의 위치를 기억하고 짝을 맞추는 메모리 게임
- 난이도에 따라 다른 규칙과 도전 요소 제공
- 시간과 시도 횟수를 최소화하여 높은 점수 달성

### 핵심 메커니즘
- **카드 매칭**: 같은 카드 2장(또는 3장)을 찾아 짝 맞추기
- **카드 미리보기**: 쉬움/보통/어려움 난이도에서 게임 시작 전 카드 공개
- **폭탄 카드**: 클릭 시 페널티 (시간 감소, 카드 섞임 등)
- **즉사, 카드 섞임**: 특정 확률로 강한 패널티 적용
- **시간 제한**: 난이도에 따라 제한 시간 설정
- **점수 계산**: 시도 횟수, 시간, 난이도 가중치를 종합한 점수

### 게임 규칙

#### 난이도 옵션
- **지옥**: 최고 난이도 (카드 수 최대, 시간 제한)
- **어려움**: 높은 난이도
- **보통**: 중간 난이도
- **쉬움**: 쉬운 난이도 (카드 수 최소)

#### 점수 시스템
- 시도 횟수에 따른 점수 계산
- 시간에 따른 보너스/감점
- 난이도별 가중치 적용

### 연출 및 인터랙션

#### 시각 효과
- 카드 호버 효과
- 클릭 시 애니메이션
- 짝 맞춤 성공/실패 연출
- 난이도 선택 시 미리보기

#### 난이도별 컨셉
- **쉬움**: 부드러운 색감, 친근한 느낌
    - 매칭 방식: 같은 카드 2장 찾기
    - 카드 구성: 총 8장 (4쌍)
    - 게임 시작: 5초간 모든 카드 공개 후 시작
    - 제한 시간: 없음 (기록 갱신 모드)

- **보통**: 밝은 색감, 균형잡힌 디자인
    - 매칭 방식: 같은 카드 2장 찾기
    - 카드 구성: 총 16장 (8쌍)
    - 게임 시작: 7초간 모든 카드 공개 후 시작
    - 제한 시간: 없음 (기록 갱신 모드)

- **어려움**: 진한 색감, 도전적인 느낌
    - 매칭 방식: 같은 카드 2장 찾기
    - 카드 구성: 총 32장 (15쌍 + 폭탄 2장)
    - 게임 시작: 카드 즉시 뒤집힘 (미리보기 없음)
    - 제한 시간: ??분
    - 특수 요소: 폭탄 카드 2장

- **지옥**: 지옥 배경, 도전자를 깔보는 느낌의 대사
    - 매칭 방식: 같은 카드 3장 찾기
    - 카드 구성: 총 48장 (16세트 + 폭탄 4장)
    - 게임 시작: 카드 즉시 뒤집힘 (미리보기 없음)
    - 엄격한 제한 시간
    - 폭탄 카드 효과:
        - 기본: 시간 대폭 감소
        - 3% 확률: 남은 카드 섞임
        - 1% 확률: 즉사
