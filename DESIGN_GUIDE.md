# 🎨 디자인 가이드

카드 매칭 게임의 디자인 시스템 및 스타일 가이드입니다.

## 🎯 디자인 컨셉

**Soft Toy / Plushie Aesthetic** - 부드럽고 귀여운 장난감 느낌

### 핵심 키워드
- 🧸 **Cute & Friendly**: 귀여운 곰 캐릭터, 둥근 모서리
- 🌸 **Pastel Colors**: 부드러운 파스텔 색상 팔레트
- ✨ **Playful**: 재미있는 애니메이션과 인터랙션
- 🎈 **Approachable**: 친근하고 접근하기 쉬운 UI

## 🖋️ 타이포그래피

### 폰트 패밀리
```css
주요 폰트: Pretendard (깔끔한 한글)
보조 폰트: Noto Sans KR (구글 폰트)
시스템: -apple-system, BlinkMacSystemFont, 'Segoe UI'
```

### 폰트 크기
```javascript
타이틀:   48px (큰 제목, 게임 이름)
버튼:     32px (난이도 선택 버튼)
UI:       24px (점수, 시간, 하트)
작은 글씨: 18px (헬퍼 메시지)
```

### 스타일 특징
- **굵은 글씨**: 타이틀과 버튼은 BOLD 사용
- **하얀 테두리**: 타이틀 텍스트에 8px 두께의 흰색 stroke
- **그림자**: 타이틀 아래에 살짝 어두운 그림자 (depth 효과)

## 🎭 UI 컴포넌트

### 1. 곰 캐릭터 (Bear Character)
레퍼런스 이미지의 베이지색 곰을 p5.js로 구현

**구조:**
- 몸통: 타원형 (세로로 약간 긴 비율 1:1.2)
- 귀: 양쪽에 작은 원 2개
- 얼굴: 중앙에 조금 작은 타원
- 눈: 점 또는 웃는 눈 (기분에 따라)
- 코: 작은 타원
- 입: 작은 아치 모양
- 볼: 양쪽에 핑크색 타원

**표정 상태:**
- `isHappy = false`: 기본 표정 (점 눈, 작은 입)
- `isHappy = true`: 웃는 표정 (반달 눈, 큰 미소)

### 2. 알약 버튼 (Pill Button)
둥글둥글한 캡슐 모양 버튼

**스타일:**
- 양쪽 반원 + 중앙 사각형 조합
- 6px 두께의 하얀 테두리
- 상단에 하이라이트 (광택 효과)
- 그림자 (offset 6px + 호버 시 증가)

**호버 효과:**
- 5px 위로 들어올림
- 그림자 확대
- 부드러운 easing (0.15 속도)

### 3. 카드 디자인
정사각형 카드 (110x110px)

**공통 스타일:**
- 둥근 모서리: 20px
- 하얀 테두리: 6px
- 그림자: 4px offset + 호버 시 증가
- 광택 효과: 상단에 하얀 타원

**뒷면 (핑크):**
- 배경: #FFB4D1
- 패턴: 4개의 작은 하트 (♥)

**앞면 (파스텔):**
- 배경: 6가지 파스텔 색상 중 하나 (ID에 따라)
- 아이콘: 이모지 (🍎🍄🚀💎🔑✉️ 등)
- 매칭 완료: 반투명 초록 오버레이 + 체크 마크

### 4. 장식 요소

**물결 (Waves):**
- 하단에 2개 레이어
- 사인 곡선으로 부드러운 웨이브
- 지속적인 움직임 애니메이션

**구름 (Clouds):**
- 3개의 구름 (크기 다름)
- 느린 속도로 이동
- 반투명 흰색

**말풍선 (Speech Bubble):**
- 둥근 사각형 + 작은 꼬리
- 캐릭터 옆에 배치
- 도움말 텍스트 표시

## ✨ 애니메이션

### 카드 애니메이션

**뒤집기 (Flip):**
```javascript
duration: 300ms
effect: 3D 회전 (X축 기준)
easing: Math.cos 기반
```

**매칭 성공 (Bounce):**
```javascript
duration: 600ms
effect: 통통 튀는 확대/축소
easing: Math.sin 기반 (3회 반복)
```

**매칭 실패 (Shake):**
```javascript
duration: 400ms
effect: 좌우 흔들림 + 약간 회전
easing: Math.sin 기반 (8회 반복)
```

### 호버 애니메이션

**카드 호버:**
- 8px 위로 들어올림
- 그림자 진해짐
- 부드러운 전환 (0.2 속도)

**버튼 호버:**
- 5px 위로 들어올림
- 그림자 확대 및 진해짐
- 부드러운 전환 (0.15 속도)

### UI 애니메이션

**메시지 표시:**
- Fade in/out (300ms)
- 박스와 함께 등장

**파티클 효과:**
- 매칭 성공 시 폭죽
- 게임 완료 시 색종이

## 🎮 인터랙션 가이드

- **기본**: `cursor: pointer`

### 피드백 시스템

**성공 메시지:**

**헬퍼 메시지:**

- 흰색 텍스트
- 상단에 4초간 표시

## ♿ 접근성

### 웹 접근성 고려사항

**색상 대비:**
- 텍스트와 배경 대비율 4.5:1 이상 준수
- 진한 파란색 (#2E5C8A) 사용

**포커스 표시:**
- 3px 두께의 파란색 outline
- 2px offset

**키보드 내비게이션:**
- ESC 키: 게임 리셋
- D 키: 디버그 모드

**스크린 리더:**
- `.sr-only` 클래스로 보조 텍스트 제공
