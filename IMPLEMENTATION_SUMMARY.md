**파일**: `js/config.js`

```javascript

    NORMAL: 'normal',   // 일반 카드
    BONUS: 'bonus',     // 정답 짝 카드 (자동 매칭)
    BOMB: 'bomb'        // 폭탄 카드 (페널티)
};
```

**특수 카드 설정**:
- **보너스 카드**: 2초 후 자동 공개, 50점 보너스
- **폭탄 카드**: 30초 시간 페널티, 카드 섞임 효과
